# MERN App

## Setup Instructions

### Backend
1. Navigate to the `backend` folder:
   1.first open terminal
   2.cd backend
   3.npm install
   4.npm run dev (to run backend server)
2. Put your mongodb URL in .env file

   
### Frontend
1. Navigate to the `frontend` folder:
   first open terminal
   cd frontend
   npm install
   npm run dev
